# TruthLens AI

TruthLens AI is a web-based application that detects whether a news article is **real or fake**. It also includes AI-powered **image and text analysis features**, making it a comprehensive tool for verifying information.

## Features

- **News Analysis**: Detect if a news article is real or fake using machine learning models.
- **Text Detection**: Analyze text content for accuracy and credibility.
- **Image Analysis**: Detect misinformation or manipulation in images.
- **User-Friendly Interface**: Simple and clean design for easy interaction.
- **Python Backend**: Built with Flask and integrated with TensorFlow and other ML libraries.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: Python, Flask  
- **Machine Learning**: TensorFlow, scikit-learn, joblib  
- **Audio & Image Processing**: librosa, OpenCV
